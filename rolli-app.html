Rolli: Overview


Rolli is an app that bridges the gap between exercise and fitness apps, designed specifically with skaters in mind. What started as a passion project has now evolved into a full-fledged development effort, complete with a design language, design system, user interface, interactive prototyping, and user testing with fellow skaters and friends.


Problem


Although there are many fitness apps available for activities like running, biking, walking, and weightlifting, there are few that are dedicated exclusively to passionate skaters. This includes roller-skating, inline skating, and skateboarding.


Goals


Create an inclusive design system that accommodates all members of the skate community by incorporating engaging and familiar elements to enhance the user experience. Integrate features that connect skaters with each other and the community.


Strategy


Create an inclusive community around all things skating. Maintain consistency in the app's interface, interactions, and elements. Use recognizable user patterns from apps already being used for activity tracking.
