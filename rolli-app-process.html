User Research


Although there are many fitness apps available for activities like running, biking, walking, and weightlifting, there are few that are dedicated exclusively to passionate skaters. This includes roller-skating, inline skating, and skateboarding.


Competitor Analysis


Analyzed the apps that I currently use on my Apple Watch, as well as fitness apps that I have used in the past. I asked my community what they use to track their skating and how it affects their skate and their experience overall using it. Many applications on the App Store currently only focus on mainstream forms of exercise, such as at-home workouts, cardio, running, weightlifting, biking, and hiking.


Wireframes


I used basic wireframing to map out the desired features, with the intention of refining them later through user testing. Ideally, I would have created a wireframe for every single feature, but given the timeframe, I had to work more quickly on this.


User Flows


I had to map out these flow charts to see exactly what the user would be doing when they use Rolli. It was a big help for me to be able to visualize everything step-by-step.


Design System


This is the design system that was created for Rolli. It creates a consistent and cohesive user experience, giving users an enjoyable time while using the app. The color palette is vibrant and energetic, reflecting the exciting nature of rollerskating. Bold and dynamic colors are used to evoke a sense of adventure and fun. The typography is legible and playful, adding to the overall lighthearted tone of the app. The consistent spacing and layout guidelines ensure that the app's interface is visually balanced and aesthetically pleasing. Additionally, accessibility is a top priority in the design system. Appropriate color contrast ratios and text sizes are used to make the app easy to use for everyone.




UI Design


My design process involves creating multiple sets of low-fidelity designs and receiving feedback from the user group. In this case, I would show the design to my mentor and the skate community, and continuously improve the visual design to ensure it matches the cohesive design system I have created. In the following images you can observe that I started out with a very colorful UI but switched it to more neutral to stick with the strategy I had developed in the beginning of the project.
